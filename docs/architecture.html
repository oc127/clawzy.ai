Layer 1
App 层 — 用户看到的一切
💬 内置聊天 CORE
App 自带聊天界面，下载即用，零配置。支持文字/图片/文件/语音，富交互卡片。
  • 主聊天通道，不依赖第三方 IM
  • 支持多 Agent 会话切换
  • 任务进度、状态实时展示
⚙️ 控制面板 CORE
Agent 管理中心：状态监控、模型切换、用量统计、技能管理。
  • 龙虾状态：在线 / 离线 / 繁忙
  • Token 用量 & 费用面板
  • 一键切换 AI 模型
🔌 扩展通道 扩展
可选连接外部 IM，让龙虾同时活在多个平台。
  • Telegram / WhatsApp / Discord
  • WeChat / Feishu / DingTalk
  • 可视化配置，无需代码
🛒 技能商店 扩展
兼容 ClawHub 生态，一键安装社区 Skills，也可上传自建技能。
  • 浏览 / 搜索 / 安装 Skills
  • 兼容 ClawHub 5000+ 社区技能
  • 用户自建技能上传分享
↕ WebSocket 实时通信
↕ REST API
⚡
Layer 2
后端服务层 — 调度中枢
👥 用户系统
注册/登录、订阅管理、权限控制
  • OAuth / 邮箱 / 手机号登录
  • 订阅计费（Stripe / 支付宝 / 微信）
  • 多档定价 & 用量限额
🧠 模型路由
统一 API 网关，对接 10+ 模型供应商，用户自选模型。
Claude GPT-4o/5 Gemini DeepSeek Qwen Doubao GLM MiniMax Mistral
  • 批量采购 API，打包进订阅
  • 用户无需自己申请 API Key
  • 智能路由：简单任务→便宜模型
🐳 容器编排
管理每个用户的 OpenClaw 实例生命周期。
  • 注册 → 自动创建隔离容器
  • 健康检查 & 自动重启
  • 资源限额（CPU / 内存 / 存储）
  • OpenClaw 版本自动更新
📊 数据层
用户数据、对话历史、计费记录持久化存储。
  • PostgreSQL — 用户 / 订阅 / 计费
  • Redis — 会话缓存 / 限流
  • OSS — 文件 / 记忆 / Agent 工作区
↕ Docker API / gRPC
↕ Agent 指令流
🦞
Layer 3
OpenClaw 运行时 — 龙虾住的地方
📦 用户容器
每个用户一个独立的 OpenClaw 实例，Docker 容器隔离运行。
  • 独立工作区 & 文件系统
  • 持久化记忆 (MEMORY.md)
  • 每日会话摘要自动生成
  • Heartbeat 心跳保活机制
🛠️ 工具引擎
OpenClaw 内置工具 + ClawHub 技能执行环境。
  • 浏览器操控 (Playwright)
  • Shell 命令执行
  • 文件读写 / 邮件 / 日历
  • ClawHub Skills 运行沙箱
🔒 安全层
容器级隔离，确保用户间零数据泄露。
  • 每容器独立网络命名空间
  • CPU / 内存 / 磁盘 cgroup 限额
  • 文件权限沙箱 (OpenClaw plugin)
  • 定期安全扫描 & 日志审计
🔄 同步管理
与 OpenClaw 上游保持同步，自动获取新功能。
  • Fork 管理，<24h 同步上游
  • 自动化测试 → 灰度发布
  • 用户无感升级（热更新）
